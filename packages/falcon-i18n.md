# Falcon i18n

This package provides default internationalization resources.

Resources are located in `./i18n` directory and are organized accordingly to `{{lng}}/{{ns}}.json` pattern. Then for English it looks like this:

```text
i18n
  |-en
    |-common.json
    |-blog.json
    |-...
  |-...
```

## Provided languages

* EN
* NL
