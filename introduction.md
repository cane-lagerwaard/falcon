# Introduction

DEITY Falcon is a platform agnostic PWA, stand-alone but modular library to be used with any e-commerce platform.

## Technologies inside

* Apollo
* NodeJS
* Razzle
* GraphQL
* React
* Koa
* Jest \(testing\)
